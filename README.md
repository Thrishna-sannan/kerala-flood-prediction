#KERALA FLOOD PREDICTION (2026)

##OVERVIEW

This project predicts flood risk in Kerala using machine learning techniques based on rainfall, river levels, and environmental factors. It includes data preprocessing, visualization, and predictive modeling

##FEATURES

* Data cleaning and preprocessing
* Visualization (scatter plots, heatmaps, histograms, etc.)
* Flood probability prediction using **Linear Regression**
* Flood occurrence classification using **Logistic Regression**
* Risk level classification: **LOW / MEDIUM / HIGH**

##TECHNOLOGIES USED

* Python
* Pandas, NumPy
* Matplotlib, Seaborn
* Scikit-learn

##DATASET

The dataset contains features like:

* Daily & Monthly Rainfall
* Soil Moisture
* River Level
* Reservoir Level
* Flood Probability Score
* Flood OccurrencE
